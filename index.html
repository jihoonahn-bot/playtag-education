연결 설정

다른 제품과 연동해서 생산성을 올려보세요.

Google 캘린더
교육 일정 캘린더
배정 확정된 교육·설명회 일정이 Google 캘린더에 자동으로 등록됩니다.
연동 중
⚙️
API 설정
Google Cloud Console에서 OAuth Client ID를 발급받아 입력해주세요.
Client ID 입력 후 적용하면 Google 계정 연결이 활성화됩니다.
로그인 허용 계정 관리
아래 등록된 Google 이메일만 관리자·담당자 탭에서 Google 로그인이 가능합니다.
담당자는 담당자 관리 탭에서 이메일 등록 시 자동으로 로그인이 허용됩니다.
관리자 이메일
💡 담당자는 별도 등록 불필요 — 👤 담당자 관리 탭에서 이메일 입력 시 자동 허용됩니다.
1
Google Cloud Console 프로젝트 생성
console.cloud.google.com → 새 프로젝트 → 이름 입력(예: playtag) → 만들기
2
Google Calendar API 활성화
API 및 서비스 → 라이브러리 → "Google Calendar API" 검색 → 사용 설정
3
OAuth 동의 화면 설정
외부 선택 → 앱 이름·이메일 입력 → 범위에 .../auth/calendar 추가 → 테스트 사용자 추가
4
OAuth 2.0 클라이언트 ID 발급
사용자 인증 정보 → OAuth 클라이언트 ID → 웹 애플리케이션 → 승인된 JS 출처에 https://[내아이디].github.io 입력 → 클라이언트 ID 복사
5
위 ⚙️ API 설정 카드에 붙여넣기
복사한 Client ID를 위 입력란에 붙여넣고 적용 버튼을 클릭합니다. gcal.js 파일의 CLIENT_ID도 동일하게 수정해주세요.
